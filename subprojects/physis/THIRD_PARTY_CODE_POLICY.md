# Third-Party Code Policy

## Rules

1. Do not copy code, models, datasets, or simulator assets without clear license terms.
2. Track source URL, license, and local modifications for every imported component.
3. Preserve all required attribution and notices.
4. Treat copyleft inputs as explicit review items before incorporation.
5. Prefer interfaces and adapters over uncontrolled code ingestion.

Open-source intent and delayed monetization do not remove license obligations.
