# Kinema Subprojects

## Titan

Role inside Kinema:

- embodied agency
- world-model architecture
- hardware envelope
- safe action and bounded autonomy

Core contribution:

Titan keeps Kinema from collapsing into a pure integration framework with no
serious thesis about long-horizon embodied intelligence.

## Physis

Role inside Kinema:

- simulator-first stack
- perception and control interfaces
- ROS runtime and deployment spine
- datasets and evaluation discipline

Core contribution:

Physis keeps Kinema from becoming a speculative robotics research shell with no
usable engineering substrate.
